## Getting Help

### Before Asking

1. **Search the docs** - Your question may be answered
2. **Check FAQ** - [Frequently Asked Questions](./faq.md)
3. **Check issues** - Someone may have had the same problem
4. **Try troubleshooting** - [Troubleshooting Guide](./troubleshooting.md)

### How to Ask

When asking for help:

1. **Describe the problem clearly**
   - What you're trying to do
   - What's happening instead

2. **Provide context**
   - db-yard version
   - Operating system
   - Deno version

3. **Include relevant logs**
   - Error messages
   - stderr output
   - Verbose mode output

4. **Show what you've tried**
   - Steps taken
   - Results of each attempt

### Where to Ask

- **GitHub Issues**: Bug reports and feature requests
- **GitHub Discussions**: General questions and ideas
- **Stack Overflow**: Tag with `db-yard` (if applicable)
